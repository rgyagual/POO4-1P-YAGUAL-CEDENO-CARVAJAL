# Sistema de Venta y Gestión de Entradas para el Mundial

## Descripción
Este proyecto es una aplicación de consola en Java que simula el funcionamiento básico de un sistema de venta y gestión de entradas para partidos del Mundial. En el sistema intervienen aficionados y organizadores del evento.

---

## Funcionalidades generales
- Inicio de sesión de usuarios
- Sistema de notificaciones

Dependiendo del tipo de usuario, se habilitan diferentes funciones:

---

## Aficionado (A)
1. Consultar partidos
2. Comprar entrada
3. Comprar kit de entradas
4. Consultar entradas

---

## Organizador (O)
1. Consultar entradas
2. Generar reporte

---

## Tecnologías utilizadas
- Java
- Programación Orientada a Objetos (POO)
- IDE: Visual Studio Code

---

## Objetivos académicos
Este proyecto tiene como objetivo fortalecer el aprendizaje desarrollando un sistema que aplique los principios fundamentales de la Programación Orientada a Objetos (POO):

- Abstracción
- Encapsulamiento
- Herencia
- Polimorfismo
- Sobrecarga
- Sobreescritura
